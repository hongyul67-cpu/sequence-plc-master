# 👩‍🏫 교사용 설정 안내 — 학생 완료 자동 기록

이 학습 게임은 기본적으로 **개인용**이에요. 학생의 진도(이어하기)는 각자 기기에 자동 저장되지만,
**교사가 "누가 어디까지 했는지" 확인**하려면 아래 한 번의 설정이 필요해요. (무료, 약 10분)

원리: 학생이 한 단계를 끝낼 때마다 → 교사의 **구글 시트에 한 줄씩 자동 기록**돼요.

---

## 1단계. 구글 시트 만들기
1. [sheets.new](https://sheets.new) 로 새 구글 시트를 만들어요. (이름 예: `시퀀스PLC 학습기록`)

## 2단계. Apps Script 붙여넣기
1. 그 시트에서 상단 메뉴 **확장 프로그램 → Apps Script** 클릭
2. 기본 코드(`function myFunction`)를 **전부 지우고** 아래 코드를 붙여넣어요:

```javascript
function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var ss = SpreadsheetApp.getActiveSpreadsheet();
    var sh = ss.getSheetByName('기록') || ss.insertSheet('기록');
    if (sh.getLastRow() === 0) {
      sh.appendRow(['시각','반','번호','이름','단계','단계명','점수','정답','총문항','최고콤보']);
    }
    sh.appendRow([data.at, data.cls, data.num, data.name, data.stage,
                  data.stageName, data.score, data.correct, data.total, data.combo]);
    return ContentService.createTextOutput('ok');
  } catch (err) {
    return ContentService.createTextOutput('error: ' + err);
  }
}
```
3. 💾 저장(Ctrl+S)

## 3단계. 웹앱으로 배포
1. 오른쪽 위 **배포 → 새 배포** 클릭
2. 톱니바퀴 ⚙️ → **웹 앱** 선택
3. 설정:
   - **실행 대상(Execute as)**: 나 (본인 계정)
   - **액세스 권한(Who has access)**: **모든 사용자(Anyone)**
4. **배포** → 권한 승인(본인 구글 계정으로 로그인, "안전하지 않음 → 계속" 눌러 진행)
5. 나오는 **웹 앱 URL**을 복사해요. (`https://script.google.com/macros/s/......../exec` 형태)

## 4단계. 게임에 URL 연결
`시퀀스_PLC_마스터.html` 파일을 편집기로 열고, 아래 줄을 찾아요(파일 위쪽 `<script>` 근처):

```javascript
let LOG_ENDPOINT = '';
```

따옴표 안에 복사한 URL을 넣어요:

```javascript
let LOG_ENDPOINT = 'https://script.google.com/macros/s/......../exec';
```

저장 후 배포(깃허브에 올리면 학생용 링크에 반영). **끝!**

---

## 확인 방법
- 학생이 게임에 들어오면 **반·번호·이름**을 한 번 입력해요.
- 한 단계를 완료할 때마다 교사 시트 `기록` 탭에 한 줄씩 쌓여요:
  `시각 · 반 · 번호 · 이름 · 단계 · 단계명 · 점수 · 정답 · 총문항 · 최고콤보`
- 직접 한 판 완료해보고 시트에 줄이 생기면 정상이에요.

## 참고 / 주의
- **개인정보**: 실명 대신 **번호나 별명**을 쓰게 하면 더 안전해요. 시트는 교사 본인 구글 계정에만 있어요.
- `LOG_ENDPOINT`를 비워두면(`''`) 기록을 전혀 보내지 않는 **개인용**으로 동작해요(이어하기만).
- 기록 전송은 조용히(백그라운드) 이뤄져서, URL이 틀려도 학생 학습은 멈추지 않아요 → 그래서 **교사가 직접 한 번 테스트**하는 게 중요해요.
- "기기 넘나들기"(학교 PC ↔ 집 폰) 이어하기는 아직 안 돼요. 같은 기기·브라우저에서는 자동 유지돼요. (원하면 추가 개발 가능)
